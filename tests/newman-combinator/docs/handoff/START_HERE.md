```json
// config/test-scenarios.json
{
  "name": "your-scenario",
  "dataDimensions": ["naughtyStrings"],
  "endpointTemplate": "/api/v1/Books/{{naughtyString}}",
  "method": "POST",
  "expectedResults": {
    "statusCode": [400, 404]
  }
}
```

### Update API Endpoint:
```json
// config/test-scenarios.json
{
  "endpointTemplate": "/api/v2/YourEndpoint/{{param}}"
```
